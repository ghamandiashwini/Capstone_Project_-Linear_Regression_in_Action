# Linear Regression in Action

## Overview

The **Linear Regression in Action** project demonstrates the application of linear regression techniques to analyze relationships between variables and build predictive models. Using Python and machine learning libraries, this project explores how linear regression can be used to predict outcomes based on historical data.

The project covers the complete machine learning workflow, including data exploration, preprocessing, model training, evaluation, and interpretation of results.

---

## Objectives

- Understand the relationship between independent and dependent variables
- Perform exploratory data analysis (EDA)
- Prepare data for machine learning modelling
- Build a linear regression model
- Evaluate model performance using regression metrics
- Interpret model predictions and insights

---

## Machine Learning Approach

The project follows these steps:

### 1. Data Exploration
- Loaded and inspected the dataset
- Analysed data structure and summary statistics
- Identified trends and relationships between variables

### 2. Data Preprocessing
- Handled missing values
- Prepared features and target variables
- Split data into training and testing datasets

### 3. Model Development
- Implemented Linear Regression using Scikit-learn
- Trained the model using training data
- Generated predictions on test data

### 4. Model Evaluation
Evaluated model performance using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## How to Run

1. Clone the repository

```bash
git clone https://github.com/ghamandiashwini/Capstone_Project_-Linear_Regression_in_Action.git
```

2. Navigate to the project directory

```bash
cd Linear_Regression_in_Action
```

3. Install required libraries

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open the notebook and run all cells.
